1. Introduction
1.1. PURPOSE
Суть документа в описании функциональных и нефункциональных особенностей приложения при контакте с пользователем.

1.2  INTENDED AUDIENCE AND READING SUGGESTIONS
Приложение предназначено для личного использования, может вестись индивидуально или совместно с кем-то.

1.3 PROJECT SCOPE
Цель приложения -- облегчить ведение и мониторинг личных финансов и создать удобное и простое в использование приложение. Система будет доступна на двух мобильных платформах. У нас будет сервер базы данных, поддерживающий аккаунты пользователей, историю операций для получения статистики. Прежде всего, мы надеемся обеспечить удобный пользовательский опыт.

2. Overall Description
2.1 PRODUCT PERSPECTIVE
Приложение содержит и отображает информацию о расходах/доходах пользователя по категориям за установленный период. Может отображать информацию списком (по наиболее затратной категории или отображать информацию на дневной основе) или пиктограммами категорий. 
Приложение может отображать информацию в двух режимах:
- с установленным лимитом средств с последующим отслеживанием остатка и "подчеркивание" того, когда лимит превышен. 
- без лимита, отображаются только затраты/доходы
Также есть возможность синхронизации, выгрузки статистики в .CSV, резервного копирования.

2.2 USER CLASS and CHARACTERISTICS
Предполагается, два типа пользователей: индивидуалы и семьи. Допускается использование приложения в маленьких компаниях, но функционала PRO-версии достаточно для удовлетворения их нужд.
Пользователи приложения могут вносить расходы/доходы по категориям. У пользователя бесплатной версии есть возможность:
- зарегистрироваться
- управлять финансами
1) вести расходы/доходы по категориям и дням с возможность записи комментария
2) видеть остаток, если он выбрал режим лимита
3) выбирать необходимый период (можно настроить кастомный период)  
4) видеть статистику за период
5) видеть статистику затрат по категориям 
6) пользоваться поиском для нахождения какой-либо записи (ищут по комментарию)
- иметь доступ к настройкам приложения
1) настраивать режим отображения баланса (лимит/безлимит) 
2) выбирать язык приложения
3) настраивать первый день недели и первый месяц года
4) экспортировать данные в формате .CSV
5) создать резервную копию данных
6) выбирать валюту приложения

В дополнение к бесплатной версии, пользователи PRO-версии могут:
- установить локальный пароль для приложения
- синхронизировать аккаунт с Google Drive, Dropbox
- иметь несколько внутренних профилей с разными валютами
- создавать собственные категории
- синхронизироваться с несколькими устройствами
- предоставлять доступ для совместного редактирования баланса

3 FUNCTIONAL REQ
Для бесплатной версии:
3.1 Создание аккаунта/регистрация (возможность регистрации через профиль гугл или создание нового аккаунта, + через AppleID для iOS. При создании аккаунта с нуля необходимо ввести почту и придумать пароль)
3.2 Настройка профиля (выбор валюты, языка приложения, периода ведения статистики)
3.3 Ведение приложения
3.3.1 запись расходов с возможностью комментировать (ввод расходов вносится через кнопку -, далее пользователь вносит сумму, выбирает категорию, на которую пошли расходы, пишет комментарий при необходимости; также есть возможность выбрать день расхода, если пользователь забыл сделать запись за предыдущий день. Альтернативный вариант внесения расходов -- клик на категорию, вводит сумму, пишет комментарий и выбирает дату при необходимости)
3.3.2 запись доходов с возможностью комментировать (при пополнении финансов пользователь нажимает кнопку +, вводит сумму, пишет комментарий при необходимости, выбирает категорию дохода: зарплата, сбережения)
3.3.3 отображение текущих расходов (пользователь видит расходы в процентном соотношении по категориям на главном экране, есть возможность просматривать расходы списком: список может быть по категориям либо по дням)
3.3.4 установка текущего периода (отчетный период задан приложением: день, неделя, месяц, год, все время либо может быть установлен вручную)
3.3.5 просмотр предыдущих периодов
3.3.6 поиск
3.3.7 отображение данных в пиктограммах в процентном соотношении расходов на ту или иную категорию на главной странице или списком с соответствующими категориями
3.4 Общие настройки приложения
3.4.1 выбор языка приложения
3.4.2 выбор первого дня недели
3.4.3 выбор первого месяца года
3.4.4 выбор режима
3.4.4' с установленным вручную лимитом
3.4.4" без лимита
3.5 Экспорт данных в формате .CSV
3.6 Создание резервной копии

Для PRO-версии
